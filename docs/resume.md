# Resume Description

**NetDiag Agent: AI Agent for Local Network Problem Diagnosis**

Built a local AI agent for diagnosing network lag in Wi-Fi, dorm, home, and office scenarios.
The system collects real network evidence from the user's computer, including gateway latency,
DNS configuration, DNS resolution time, packet loss, and optional route tracing. Designed a
symptom-aware planning module that selects different workflows for gaming lag, web access
issues, single-site slowness, and deep route diagnosis. Implemented rule-based root-cause
analysis, DeepSeek-based tool planning with allowlisted local probes, Agent Trace visualization,
and short monitoring for intermittent packet loss and jitter. Integrated DeepSeek API to
generate evidence-grounded reports for users, network administrators, or ISP support.

**Tech stack:** Python, Streamlit, DeepSeek API, pandas, Windows network tools, pytest.
